# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities privately.

Contact: security@colleagueai.ai

Please include:

- Affected URL or component
- Steps to reproduce
- Potential impact
- Screenshots or proof of concept, if safe to share
- Your contact details for follow-up

## Scope

In scope:

- colleagueai.ai public website
- Checkout and entitlement flows
- Partner attribution flows
- Public API endpoints owned by ColleagueAI

Out of scope:

- Denial-of-service testing
- Social engineering
- Physical attacks
- Attacks against third-party systems not controlled by ColleagueAI

## Responsible Testing

Do not access, modify, delete, or exfiltrate user data. If you encounter sensitive data, stop testing and report immediately.
